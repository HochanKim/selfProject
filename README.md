# 영상컨텐츠 제작회사의 홈페이지 구성하기 (Spring Boot, Maven)

+  Spring 프레임워크에서 MVC 패턴을 활용하여 영상컨텐츠 회사의 웹 화면 구성, MySQL에 저장한 데이터들을 이용하여 CRUD를 적용
